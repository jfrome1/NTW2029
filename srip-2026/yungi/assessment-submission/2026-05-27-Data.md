# Analysis of what data we have (PostHog AI)

<h3>What data exists</h3>

- Behavioral data from NTW course website over roughly one year (May 2025 ~ May 2026)

## Standard PostHog events (auto-captured)

| Event       | Volume | Unique users | Notes |
|-------------|--------:|--------------:|-------|
| `$autocapture` | 30,321 | 340 | Every click, input, and change on the page |
| `$pageview`    | 15,891 | 934 | Each page load |
| `$pageleave`   | 14,265 | 966 | Exit from a page |
| `$rageclick`   | 218 | 57 | Rapid repeated clicks – potential UX friction |

## Custom navigation/interaction events

| Event | Volume | Unique users | Key properties |

| `externalLinkClick` | 283 | 32 | `link`, `text` |
| `openNutshell` | 5,220 | 117 | `text`, `page` |
| `inactiveNutshell` | 3,439 | 108 | `page` |
| `closeNutshell` | 2 | 2 | `page` |

## Custom engagement/attention events

| Event | Volume | Unique users | Key properties |
|---|---:|---:|---|
| `read` | 7,589 | 251 | `page` (heartbeat every ~15 sec of scrolling/reading) |
| `tabFocused` | 77 | 37 | `timeFocused`, `timeHidden`, `totalTimeOpen`, `page`, `isInitial` |
| `tabUnfocused` | 59 | 27 | Same as above |


1) read (7,589 events) : our proxy for time-on-page.
2) $pageview + internalLinkClick : navigation pathways
3) openNutshell / inactiveNutshell : content curiosity

# Using seqMAC Method to test it may be helpful

<h3>Methodologies</h3>

<h4>Everything was done using Python</h4>

- I made it so that the asynchronous PostHog event logs became a strict, time-ordered learning paths for every individual student (sequencing)
- stochastic mapping : instead of total clicks, I calculated the probability of a student transitioning between any two specific states (such as the likelihood of going from a Rage Click to Opening Nutshell).

- This transformed erratic web-surfing into a structured, computatble matrix.

- Then, I used machine learning and SHAP to not only predict student successs, but also forcing it to reveal exactly which specific navigationalk transitions drove its predictions.

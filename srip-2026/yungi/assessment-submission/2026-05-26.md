# General Thoughts / Questions about using Markov Chain for Student Behavior Analysis

- Main doubt / question from me : I understand how we can generate a transition matrix that describes the probabilities of which another state "a" student is going to transition to. But, my question is, "just because one student had this transition, what makes us generalize that other students would go through that path too?"

- In addition, let's say we can generalize by counting the frequency of a particular path of transition between states. What makes us say that the path would be the most "optimal" path for success? Simply put, how do we know that path is the best one for students to learn the contents "most" effectively?

- What are the factors for "successful" learning in an educational environment?

# SRIP 2026 Workspace

This folder is the shared workspace for the NUS College SRIP 2026 project, "Learning Analytics in Practice: Student Website Usage and Course Outcomes." Two interns (Yungi and Kristen) and Jonathan (supervisor) work here over the summer 2026 internship period. Joe Klaus (joeyk2k@gmail.com), the developer who built the website and set up the analytics, is available to answer technical questions.

The project's framing question is how we can use data about student use of a course website to further educational goals, i.e. which parts of the site students actually use, how that relates to course outcomes, and how the site, the data we collect, or the course materials themselves could be designed differently to better support learning.

The work has two related parts. The first is analyzing the event data PostHog collected from students using the NTW2029 course website in academic year 2025-26. If the first part is completed, we will continue to the second part, which is designing improvements in the website and data collection.

This folder lives inside the NTW2029 repo and will be the central location for our shared documents. The repo's root `README.md` documents the website itself; this README orients you to the SRIP-specific work.

## What's in this folder

- `references/` — orientation and reference documents for the project
- `meeting-notes/` — one file per meeting, named in YYYY-MM-DD format (e.g., `2026-05-12.md`)
- `yungi/` and `kristen/` — personal subfolders for each intern's working documents

We will add more folders as needed.

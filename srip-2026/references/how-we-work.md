# How we work

How the daily and weekly work is organized: meetings, reporting, communication, and where files live.

## Daily meetings

- Every weekday. Time TBD.
- Each meeting ends with clear next-step actions. If they aren't clear, ask before we close.

## End-of-day reports

- Each weekday, commit a markdown file at `[your-folder]/daily-reports/YYYY-MM-DD.md` that includes:
  - What you planned to work on today
  - What you actually worked on/got done
  - Current obstacles, questions, or things you need guidance on
  - The one or two things you plan to start with tomorrow

## Communication channels

- Email is the default. CC the other intern.
- Telegram only for things that need a quick response.
- Questions outside meetings are welcome. Try things and look things up first.

## Working files

- Everything in markdown: drafts, analyses, peer comments, daily reports.
- Save your work in your personal folder in the shared workspace so progress is visible.
- Files use lowercase with hyphens between words (e.g., `literature-review.md`, not `Literature Review.md`).
- See the workspace README for the folder layout (`meeting-notes/`, `references/`, your personal folder, and so on).
